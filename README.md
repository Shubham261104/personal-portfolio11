# Portfolio

A fast, responsive, and accessible personal portfolio website.

## Usage

- Open `index.html` in a browser, or serve the folder with any static server.
- Edit content in `index.html` (name, bio, links, projects).
- Tweak styles in `styles.css` and behavior in `script.js`.

## Features

- Responsive layout: hero, about, skills, projects, contact
- Theme toggle with persistence
- Smooth scrolling, mobile nav, simple project filters

## Deploy

Deploy to any static host (GitHub Pages, Netlify, Vercel).

- GitHub Pages: push repo and enable Pages on main branch root
- Vercel/Netlify: import repo and deploy as a static site
